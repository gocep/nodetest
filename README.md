# Sample nodejs application

Build a CRUD API for a simple application using Node.js, Express and jsonplaceholder

## Steps to Setup

1. Install dependencies

```bash
npm install
```

2. Run Server

```bash
node server.js
```

You can browse the apis at <http://localhost:3000/api>
